# To-Do List Website

A simple and responsive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.
This application allows users to add, manage, and remove tasks while automatically saving them in the browser using **Local Storage**, so tasks remain even after refreshing the page.

---

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Tasks automatically saved using **Local Storage**
* Tasks remain after page refresh
* Simple and clean user interface
* Dark Mode
* Lightweight and fast
---

## Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript** – Application logic and task management
* **Local Storage API** – Stores tasks in the browser

## How It Works

1. Enter a task in the input field.
2. Click the **Add Task** button.
3. The task appears in the list.
4. Tasks are stored in **Local Storage** so they remain saved even after refreshing the page.
